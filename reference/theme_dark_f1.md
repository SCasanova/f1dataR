# Dark F1-style Theme for ggplot

Theme for all f1dataR plot functions. Mimics Formula 1 style.

## Usage

``` r
theme_dark_f1(axis_marks = FALSE)
```

## Arguments

- axis_marks:

  True or false, whether axis line, ticks and title should be shown or
  not. Defaults to false

## Value

A ggplot object that indicates grand prix, driver, time and selected
color variable.
